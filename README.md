# s3-demo-go

## REST/API методы

    GET /create
        Описание: Загружает сжатые на 75% картинки в MinIO
        На вход: ничего
        На выходе: сообщение отправки/не отправки

    GET /get
        Описание: Загрузка все картинки на локальную папку докера в определенную папку
        На вход: ничего
        На выходе: сообщение отправки/не отправки
